# 파주인쇄소공인특화지원센터 (Paju Printing Center)

파주인쇄소공인특화지원센터의 공식 웹사이트입니다.

## 기술 스택
- **Framework**: Next.js 15 (App Router)
- **Database**: Supabase (PostgreSQL)
- **ORM**: Prisma
- **Authenticaton**: JWT (jose)
- **Deployment**: Vercel

## 주요 기능
- 메인 롤링 배너 및 상단 대시보드
- 공지사항 및 센터소식 관리
- 문의하기 및 회원가입 신청
- 관리자 전용 대시보드 및 업로드 시스템
